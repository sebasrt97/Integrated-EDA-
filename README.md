# Integrated-EDA

## Descripción del Proyecto

Este proyecto de Análisis Exploratorio de Datos (EDA) investiga la relación entre los hábitos de juego y diversos indicadores de bienestar psicológico y físico (sueño, ansiedad, aislamiento social).

La particularidad de este estudio es que enriquece los datos de encuestas personales con métricas reales de la industria obtenidas a través de la API de SteamSpy, permitiendo contrastar la percepción del jugador con el comportamiento masivo del juego que consume.

## Fuentes de Datos

Dataset Primario: Gaming and Mental Health.csv (1000 registros, 27 variables). Incluye datos sobre:

Horas diarias de juego y años de experiencia.

Indicadores de salud: Calidad de sueño, estado de ánimo, dolor físico.

Riesgo de adicción y aislamiento social.

Dataset de Enriquecimiento: SteamSpy API.

Métricas añadidas: Horas de juego promedio de la comunidad (average_forever), estimación de dueños (owners) y etiquetas de género adicionales.
